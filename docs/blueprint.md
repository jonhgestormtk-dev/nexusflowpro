# **App Name**: NexusFlow Pro

## Core Features:

- Client & Contract Lifecycle Management: Centralized management of client profiles, including all associated recurring contracts, their status (active, overdue, suspended, canceled, finished), and renewal tracking.
- Automated Recurring Invoicing: Automatically generate and track recurring invoices based on contract terms, managing payment statuses (pending, paid, overdue) and linking to clients/contracts.
- Real-time Performance Dashboard: An intuitive dashboard displaying key business metrics in real-time, such as total contracts, active clients, monthly revenue, pending payments, and upcoming renewals.
- Secure Document Storage & Retrieval: Facilitate secure upload, storage, and retrieval of contract PDFs and other relevant client documents via Firebase Storage, integrated with client and contract records.
- Role-Based Access Control & Auth: Implement secure user authentication via Firebase Authentication with email/password login, password recovery, and fine-grained access control based on user roles (Admin, Financeiro, Comercial).
- AI Contract Detail Extractor: An AI tool to intelligently parse uploaded PDF contracts, extracting essential details like service type, value, dates, and terms, to automatically populate contract creation forms.

## Style Guidelines:

- Color scheme: Dark. Primary color: A sophisticated indigo-purple (#6033CC), chosen for its modern, premium feel in tech. Background color: A deep, muted grey with an indigo undertone (#1A1A1E), providing an elegant, uncluttered backdrop. Accent color: A vibrant electric blue (#2AA1F0) to highlight interactive elements and draw attention to key data, creating strong contrast and a technological edge.
- Headline and body font: 'Inter' (sans-serif), selected for its modern, objective, and highly legible appearance across all scales, fitting a professional SaaS. Code font: 'Source Code Pro' (monospace sans-serif) for displaying any code snippets or technical data with clarity.
- Utilize 'Lucide Icons' as requested, ensuring all icons are line-style, clean, and minimalist to align with the technological and elegant aesthetic. Icons should be scalable and used judiciously to enhance navigation and convey information efficiently without clutter.
- Responsive SPA layout featuring a collapsable sidebar for navigation. The main content area will consist of modern dashboards with elegantly designed data cards, advanced interactive tables, and intuitive data visualization graphs, creating a premium and highly usable experience.
- Subtle and fluid animations, powered by Framer Motion, will be used for state transitions, interactive elements, and loading indicators. Animations will enhance user experience by providing smooth visual feedback and guided navigation without being distracting.
